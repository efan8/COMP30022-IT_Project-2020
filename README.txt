--- Friday Slacker's IT Project v1 ---
Description: Upload family artifacts
and access them on the internet.

Installation Notes:

1. Install node js v10.16.3 and npm v 6.9.0
2. Navigate to IT_Project\client in cmd
3. run npm install
4. Navigate to IT_Project folder
5. Install the following packages in IT_Project folder

npm install --save react-router-dom
npm install react-date-picker
npm install formidable
npm install browser-image-compression
npm install concurrently
npm install cookie-parser
npm install firebase-admin
npm install react-navigation

6 to run type "npm start"
